消息通道配置说明

当前版本优先支持手动添加与应用。飞书扫码自动创建应用/机器人能力暂未开放，本页仅提供各通道的字段说明和接入提示。

企业微信

需要填写 botId 与 secret。保存后会立即启用该通道。

  • botId：企业微信机器人或应用对应的标识。
  • secret：企业微信侧签发的密钥。

钉钉

需要填写 clientId 与 clientSecret。保存后会立即启用该通道。

  • clientId：钉钉应用的 Client ID。
  • clientSecret：钉钉应用的 Client Secret。

QQ

需要填写 appId 与 appSecret。保存后会立即启用该通道。

  • appId：QQ 应用标识。
  • appSecret：QQ 应用密钥。

元宝

需要填写 appId 与 appSecret。保存后会立即启用该通道。

  • appId：元宝应用标识。
  • appSecret：元宝应用密钥。

飞书

需要填写 appId、appSecret，并可选开启 Streaming 模式。保存后会进入“已配置，待配对”或“已接入”状态。

  • appId：飞书企业应用的 App ID。
  • appSecret：飞书企业应用的 App Secret。
  • 完成保存后，仍需在飞书侧拿到 pairing code，再执行 openclaw pairing approve feishu CODE。
